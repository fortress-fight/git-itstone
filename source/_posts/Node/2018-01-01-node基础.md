## 实例

1. 启用一个服务器，并输出一个 `<p>HelloWorld</p>`

    建立文件 learn01.js

    ```js learn01

        var http = require('http');

        var serv = http.createServer(function (req, res) {
            res.writeHead(200, {'Content-Type': 'text/html'});
            res.end('<p>HelloWorld</p>');
        })

        serv.listen(3000);    
    ```

    执行 `learn01.js`

    ```base
        node learn01.js    
    ```

    打开 `localhost:3000`, 我们就可以看到页面
